# arbitrage_terminal
Арбитражный торговый терминал в связке с QUIK

## Установка Lua-скриптов в QUIK

Для работы приложения необходимо установить Lua-скрипты в терминал QUIK:

1. Скопируйте содержимое папок `infra/quik/lua/` и `infra/quik/socket/` в директорию терминала QUIK
2. В файле `config.json` измените адрес для удалённого подключения (если требуется):
3. Запустите скрипт `QuikSharp.lua` в терминале QUIK
